# questionnaire-survey 问卷调查系统

## 介绍

这是一个使用 vue3 + Typescript + Vite+ant-design-vue 编写的一个问卷调查系统。本系统实现了问卷组装、答题以及统计。

## 项目设置

```sh
pnpm install
```

## 编译和热重新加载以进行开发

```sh
pnpm dev
```

## 类型检查、编译和最小化生产

```sh
pnpm build
```
